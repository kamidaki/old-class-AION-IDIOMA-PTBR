Materiais necessários para aprimoramento de armas

Para alcançar o ambiente correto para a temperatura, use o fluxo de Jotun com um agente de refino especial.

NOTA-Os agentes refinados derivados de Arachnas são os melhores. Esteja ciente de que isso será venenoso.

Nota 2-O fluxo é opcional, mas dobra o poder efetivo do agente de refino.

Sequência de temperamento

1. Defina o forno para o 2º grau alto.

2. Use um fluxo de Jotun e metade do agente de refino especial.

3. Aguarde o fluxo derreter e produza vapor.

4. Adicione o restante do agente de refino especial.
